GГўrleanuвЂ“Panageas (2015): heterogeneous agents

This is the package's most demanding kind of problem: a general-equilibrium model with endogenous volatility and several coupled value functions. Two types of agents differ in risk aversion and elasticity of intertemporal substitution вЂ” type A is "bold" (low $\gamma_A$), type B "conservative" (high $\gamma_B$). The single state $x$ is the consumption share of type A. Four claims are priced jointly: the wealthвЂ“consumption ratios $p_A, p_B$ of the two agents and two human-capital claims $\phi_1, \phi_2$.

What makes it hard is that the volatility of the state, $\sigma_x$, is endogenous вЂ” it depends on the very derivatives $p_A', p_B'$ we are solving for вЂ” so the upwind direction must be recomputed from the resulting drift $\mu_x$ each iteration.

The model

The state space

We build the grid and the initial guess first, because they fix the names used everywhere else. The grid is a NamedTuple whose key is the state variable (x, the consumption share of type A); the guess is a NamedTuple whose keys are the unknown functions (pA, pB, П•1, П•2), holding one starting value at each grid point. These names are what reappear inside the equation below вЂ” e.g. pAx_up will be the forward finite difference of pA in x.

One state $x \in [0, 1]$, four coupled unknowns, all initialized flat at one.

The equation

We now write the function encoding the equilibrium conditions. Following the package convention, it takes the current state (a grid point) and u вЂ” the local bundle holding each unknown and its finite-difference derivatives there вЂ” and returns the time derivative of each unknown (pAt, pBt, П•1t, П•2t).

Because $\sigma_x$ is endogenous, the loop recomputes the upwind direction from the drift $\mu_x$: it starts with forward differences (*_up) and switches to backward (*_down) when the drift turns negative.

    О“ = 1 / (x / ОіA + (1 - x) / ОіB)
    p = x * pA + (1 - x) * pB

    ПѓpB = pBx / pB * Пѓx

    ОјpB = pBx / pB * Ојx + 0.5 * pBxx / pB * Пѓx^2
    ОјП•1 = П•1x / П•1 * Ојx + 0.5 * П•1xx / П•1 * Пѓx^2

The solution

We saved the equilibrium interest rate $r$ and the price of risk $\kappa$ on the grid. The price of risk is high when the conservative agents (type B, high $\gamma$) hold most of the economy's wealth (low $x$) and falls as the bold agents (type A) come to dominate вЂ” risk is borne more willingly when the less risk-averse agents are wealthier.
