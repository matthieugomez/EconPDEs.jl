EconPDEs.jl

EconPDEs.jl solves nonlinear ODEs and PDEs that arise in economic models, especially Hamilton–Jacobi–Bellman equations. You write the local equation; the package supplies the finite-difference derivatives, upwinding, sparse Jacobians, and pseudo-transient Newton iteration.

Repository: github.com/matthieugomez/EconPDEs.jl

Installation

] add EconPDEs

Getting started

Start with Getting started, which solves a first model end to end. The rest of the Manual covers the package's conventions one concept at a time: naming and return conventions, upwinding, boundary conditions, time-dependent problems, and the solver and its troubleshooting. Why EconPDEs explains what this package does that general PDE software does not. The pdesolve reference lists every solver option.

The Examples in the sidebar each solve a model, plot the solution, and explain what it means economically. They are grouped by field — consumption–saving, asset pricing, corporate finance — and ordered roughly from the simplest to the most involved. Start with Neoclassical growth.
